# PETR1128
This program is a scientific calculator with many different features such as brackets, trigonometric functions, and constants such as pi and e

# Features
1. Addition
2. Subtraction
3. Multiplication
4. Division
5. Brackets
6. Sin / Asin
7. Cos / Acos
8. Tan / Atan
9. Sqaure root
10. Power of
11. Log
12. Pi
13. Eulers Number (e)
14. Percentage
15. Decimal
16. Memory Add
17. Memory Recall
18. Memory Delete
19. 0 - 9
20. Clear all
21. Equals
